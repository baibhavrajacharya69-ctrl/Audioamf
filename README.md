🔊 Audio Amplifier PCB (TDA7297 Based)
📌 Overview

This project is a compact stereo audio amplifier PCB based on the TDA7297 IC, designed for delivering high-quality sound with simple circuitry and efficient performance. It is ideal for DIY audio systems, small speakers, and educational electronics projects.

The design focuses on:

Simplicity
Reliability
Clean audio output
Easy assembly

⚙️ Features
🎵 Stereo audio output (2-channel)
🔊 Output power: ~15W + 15W (depending on supply)
🔌 Wide input voltage range (typically 6V–18V DC)
🎚️ On-board volume control (potentiometer)
💡 Power indication LED
❄️ Heat sink support for thermal management
🧩 Screw terminals for easy connections
🧠 Working Principle

The TDA7297 is a dual bridge amplifier IC. It amplifies low-power audio signals from sources like:

Mobile phones
Laptops
MP3 players

The signal path:

Audio input → coupling capacitors
Amplification via TDA7297
Output to speakers

The onboard potentiometer controls the input signal level, thus adjusting volume.
